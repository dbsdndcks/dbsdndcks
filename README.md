### Introduction
---
높은 책임감으로 최적의 코드를 생산하는 백엔드 개발자, 윤웅입니다.

복잡한 요구 사항을 적절한 책임 분리, 추상화를 통해 간결한 로직으로 구현합니다.
객체지향적이며 확장성 있는 코드 설계를 지향합니다. 가독성 높고 유연한 코드를 선호합니다.

다수의 프로젝트, 교육 과정을 통해 커뮤니케이션 능력을 강점으로효과적인 협업 능력을 갖췄습니다.
사용자 중심 사고로 개발에 임하며, 단순 기능 구현을 넘어 실질적인 문제 해결을 초점을 맞춥니다.
지속적인 학습을 통해 새로운 기술을 습득하고 기존 구조를 개선하는 데 관심이 많습니다.
